## シーン詳細

### 第一幕：出会いと違和感（Scene 1〜5）

#### SC-01: 透也の日常（消失）
- **視点**: 透也
- **内容**: 教室での出来事。消えたい時に消えず（指名される）、消えたくない時に消える（友人に無視される）。
- **恐怖**: 「自分がいなくなる」感覚。
- **きっかけ**: ネット掲示板で「制御できない能力者の集まり」を見つける。

#### SC-02: 廃ビルへの集結
- **視点**: 客観 → 透也
- **内容**: 廃ビルの一室に集まる7人。
- **初対面**: それぞれが「能力」を見せ合うが、どれも微妙に不発・暴発する。
  - 詩織：魅了しようとして、透也が引く。
  - 誠一：誰かの嘘を指摘して空気が凍る。
  - 遥人：遅刻してないのに「遅刻だ」と言われる（見えてなかった）。

#### SC-03: 共通点の確認
- **視点**: 蓮
- **内容**: 蓮がリーダーシップを取り、状況を整理する。
- **結論**: 「全員、制御できていない」「能力というより呪いだ」。
- **伏線**: 朝霧だけが沈黙している。陽菜が「後回し」能力で無邪気に遊ぶ。

#### SC-04: AC日翌週の違和感
- **視点**: 誠一
- **内容**: 7人が集まったことでDIが上昇。
- **現象**: 時計がずれる、窓の外の風景が一瞬歪む。
- **誠一の気づき**: 「見たくないもの」が見え始める。

#### SC-05: 解散と兆候
- **視点**: 透也
- **内容**: 帰り道、自分のスマホの写真フォルダから、昔の写真が消えていることに気づく。
- **引き**: 「俺たちの何かが、削られている？」

### 第二幕：探求と脅威（Scene 6〜10）

#### SC-06: 仮説の検証
- **視点**: 遥人
- **内容**: 「感情」「場所」などトリガーを探る実験。
- **結果**: 全て失敗。遥人が「消えかけ」て、みんなが彼の名前を忘れかける危機。
- **恐怖**: 遥人「俺を忘れないでくれ！」 → みんな「誰のことだっけ？」 → 「あっ、遥人！」

#### SC-07: 衝撃との遭遇（Antagonist 1）
- **視点**: 複数（アクション）
- **内容**: 街でビル倒壊事故。**BL-0041（衝撃）**が暴れている。
- **対比**: 彼は能力を誇示するが、身体が透けている。
- **結末**: 0041は「俺は最強だ！」と叫びながら、世界から「災害」として処理され消滅（IE）。
- **理解**: 「使いすぎると、ああなる」。恐怖の共有。

#### SC-08: 派閥の接触（R派・A派）
- **視点**: 朝霧 / 透也
- **内容**:
  - 朝霧にR派（江藤）が接触。「BL-0031Bを取り戻さないか？」 → 朝霧拒絶。
  - 透也にA派（桐生）が接触。「君は消えかけている。助けが必要だ」 → 透也混乱。
- **情報**: 組織の存在が明るみに出る。

#### SC-09: C派の警告
- **視点**: 蓮
- **内容**: 蓮にC派（田村）が接触。「保護」を申し出るが、蓮は拒否。
- **警告**: 「あの少女（陽菜）は危ない。遅延はいつか必ず清算される」。

#### SC-10: 変化の加速
- **視点**: 詩織
- **内容**: 詩織の周囲で依存が異常化。友人が自傷行為をして気を引こうとする。
- **絶望**: 「能力なんて欲しくなかった」。7人が再集結する。

### 第三幕：真実と崩壊（Scene 11〜15）

#### SC-11: 朝霧の告白
- **視点**: 朝霧（主観）
- **内容**: 廃ビルでの重い沈黙。朝霧が真実（PL定義）を語る。
- **告白**: 「私たちは能力者じゃない。世界に書き換えられる側」。AC-0031の真実。

#### SC-12: 狩人の襲来（Antagonist 2）
- **視点**: アクション
- **内容**: 壁が切断され、**BL-0016（切断）**が侵入。R派の実力行使。
- **戦闘**: 物理攻撃無効。透也が「認識から消える」ことで撹乱し、遥人が仲間を隠す。
- **決着**: 蓮が彼を「見ない（確定させない）」ことで、切断を無効化。0016は自己矛盾で自滅。

#### SC-13: 陽菜の崩壊
- **視点**: 陽菜 → 蓮
- **内容**: 戦闘後、安心した瞬間に陽菜に「一年分の遅延」が直撃。
- **描写**: 身体がひび割れ、存在が崩壊しかける。激痛と絶叫。

#### SC-14: 蓮の決断（代償）
- **視点**: 蓮
- **内容**: 陽菜を救う唯一の方法。「無事な姿」を現実として確定させること。
- **代償**: 膨大なDEコスト。蓮はサングラスを外し、陽菜を凝視する。「俺が見てやる」。
- **結果**: 陽菜は助かるが、蓮の左目が機能を失う（世界に徴収される）。

#### SC-15: 終結
- **視点**: 透也
- **内容**: 傷ついた7人。能力は消えない。世界は変わらない。
- **変化**: しかし、「わけがわからない」状態からは脱した。覚悟のような連帯感。
