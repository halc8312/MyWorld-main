# 社会構造（Society Structure）

## 概要

本世界の社会は、年次付与（AC）と存在上書き（IO/IE）の存在を**公式には認識していない**。
しかし、現象の影響は社会の隅々に及んでおり、都市伝説、非公式制度、産業として表出している。

---

## 1. 13歳台帳制度

### 1.1 公式制度（存在しない）

- 政府や行政による「13歳の特別管理」は公式には存在しない
- 年次付与を国家が認識しているかは**未確定**（物語ごとに調整可能）
- 仮に認識していても、世界の収束が「公式化」を阻む可能性がある

### 1.2 非公式台帳（実在する）

| 種類 | 管理者 | 内容 | 目的 |
|------|--------|------|------|
| **学校内台帳** | 教師、カウンセラー | 問題児、成績上位、孤立者のリスト | 「観察が必要な生徒」として |
| **塾・習い事台帳** | 塾経営者、コーチ | 家庭事情、連絡先、SNSアカウント | マーケティング名目 |
| **行政の断片** | 児童相談所、医療機関 | 住民票、医療記録、出欠記録 | 福祉・医療目的（断片的） |
| **綴合の統合台帳** | 綴合（Rec班） | OSINTと現場で統合した「今年の13歳候補群」 | 保持者候補の追跡 |

> 物語的に便利なのは「公式に禁止されているはずの台帳が、別の名目で実在する」こと。

### 1.3 13歳に関する社会の態度

- 一部の親は AC Day 前後に子どもを休ませる
- 「13歳の誕生日は家で静かに過ごす」という家庭内ルールが一部に存在
- 学校では「13歳検診」という名目で心理検査が行われることがある（表向きは発達検査）
- SNS では「#13歳」「#付与日」などのハッシュタグが都市伝説として流通

## 2. 付与日（AC Day）の文化

### 2.1 都市伝説としての認知

一般社会は年次付与を「都市伝説」として認識している：

| 認知の形 | 内容 |
|----------|------|
| **付与日予言** | 「今年は○月○日に能力者が生まれる」という予言サイト |
| **付与日配信** | 13歳を追跡するストーカー的な配信者（違法・炎上しやすい） |
| **付与日占い** | 「あなたの13歳の誕生日は付与日でしたか？」という占いコンテンツ |
| **13歳の呪い** | 「13歳に近づくと不幸になる」という迷信 |

### 2.2 学校・家庭の反応

- AC Day 前後は欠席率が微増する（ただし統計的には「インフルエンザの季節」等で説明可能）
- 一部の学校では「13歳記念行事」を避ける傾向がある（理由は曖昧）
- 保護者の中には「うちの子は大丈夫」と否認する層と、過剰に防護する層がいる

### 2.3 綴合の利用

- 綴合は AC Day に合わせて「回収班」を動かす
- 都市伝説の流通は DI を上げるが、「弱い観測」の洪水でもあるため、強い観測を避けやすくなる
- 綴合自身が都市伝説を流すこともある（ただしやりすぎると DI が暴発）

## 3. 保持者を巡る産業

年次付与の存在を「信じている」層により、非公式な産業が形成されている：

| 産業 | 内容 | 顧客 |
|------|------|------|
| **護衛サービス** | 13歳の子どもを AC Day 前後に護衛する | 富裕層の親 |
| **保険商品** | 「13歳リスク保険」（実際は医療保険のバリエーション） | 保護者 |
| **監視サービス** | 子どものSNS・行動を監視する | 過保護な親 |
| **隔離施設** | 「安全な合宿」として13歳を AC Day に隔離する | 信じている親 |
| **偽装身分** | 13歳の子どもの記録を曖昧にする | 犯罪者、綴合 |
| **戸籍ビジネス** | 13歳の時期をずらすための戸籍操作 | 犯罪者、富裕層 |

### 3.1 宗教的解釈

一部の宗教・新興宗教は保持者を特別な存在として解釈する：

| 解釈 | 内容 |
|------|------|
| **聖者** | 「選ばれた存在」として崇拝 |
| **罪人** | 「呪われた存在」として排除 |
| **生贄** | 「世界の均衡を保つための犠牲」として儀式化 |
| **綻び** | 「世界の欠陥が人の形で現れたもの」として観察対象 |

### 3.2 犯罪

保持者や候補を狙った犯罪も存在する：

| 種類 | 内容 |
|------|------|
| **拉致** | 保持者を捕獲し、PL を利用しようとする |
| **強制契約** | 保持者に「能力」の使用を強制する |
| **炎上誘導** | 意図的に DI を上げて現象を誘発する |
| **偽保持者詐欺** | 「自分は能力者だ」と偽り、金銭を騙し取る |

## 4. 保持者の法的地位

### 4.1 現状（未確定→確定）

> [!IMPORTANT]
> 本ドキュメントで以下を正史として確定する。

- **公式には「保持者」という法的カテゴリは存在しない**
- PL の行使は「超常現象」として法的に定義されていない
- IO/IE による被害は「原因不明」「事故」「失踪」として処理される
- 保持者自身が PL の存在を主張しても、証拠が収束で曖昧になるため、法的に認められない

### 4.2 実務上の対応

- 保持者が PL を行使した結果として起きた事象は、**最ももっともらしい説明**で処理される
- 例：「彼が来ると事故が減る」→「運が良い人」として扱われる
- 例：「彼女と話すと記憶が曖昧になる」→「心理的問題」として医療に回される
- 法執行機関は「説明できない事件」を嫌い、収束に加担する傾向がある

### 4.3 綴合の関与

- 綴合は保持者関連の事件を「処理」することで、法的な面倒を回避する
- 工作班（Ops）が「もっともらしい説明」を作り、記録を整える
- 結果として、保持者は法的には「普通の人」として扱われ続ける

## 5. 情報と観測

### 5.1 現代の観測環境

| 観測の種類 | 強度 | 特徴 |
|------------|------|------|
| 防犯カメラ | 中 | 多いが分散、管理者が多様 |
| スマホ写真 | 弱 | 編集可能、単独証言 |
| SNS投稿 | 弱 | 偽情報と混在、合意形成に影響 |
| 位置情報 | 中 | 継続的だが欠損しやすい |
| クラウド | 中 | 改ざん耐性はあるが、アクセス制御による |
| 学校記録 | 中〜強 | 公的だが更新頻度低い |
| 医療記録 | 強 | 改ざん耐性高、ただし断片的 |
| 裁判記録 | 強 | 最も強い観測、ただし作成に時間がかかる |

### 5.2 世界の収束と観測

- 世界は**弱い観測から先に改稿する**
- 強い観測には抵抗が起きるため、上書きは「別の層（関係・内面）」に逃げる
- または、強い観測そのものを「弱体化」する方向（紛失、火災、手続きミス）へ収束する
- 結果として、**記録は守れるが、人が壊れる**という逆説が生じる

### 5.3 SNS と収束

- SNS の集合知は「真実」ではなく「合意」を作る
- 世界は合意へ寄せて整合を取れるため、SNS は収束の道具になる
- ただし、SNS 上の情報過多は DI を上げ、予測不能な収束を誘発する
- 綴合は SNS を監視し、「能力者コミュニティ」の言説を利用することもある
