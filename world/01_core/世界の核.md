# 世界の核（World Core）

本ドキュメントは、創作プロジェクト共通世界設定の**中核要約**である。  
詳細は各仕様書（v1.0〜v1.3）を参照のこと。

---

## 1. 世界の根本原理（1ページ要約）

> **この世界では、現実は「物理法則」だけでなく「整合性」によって維持される。**

- 矛盾が一定閾値を超えると、世界は自律的に**整合性を保つ方向へ収束**する。
- 収束には**歪みエネルギー（DE）**が必要であり、不足すると**歪債（DD）**として蓄積される。
- 歪債は、物理現象より安価な**人物の定義（名前・関係・履歴・性格・存在）**を編集して回収される。
- ある事故（**歪む実験**）により、編集権限が一個人（**絶対的存在 / A0**）に集約された。
- 以後、A0は年に一度、13歳の人間へ**個人世界法則（PL）**を付与する。
- 保持者はPLを「**能力**」と誤認し、使うほど歪債が溜まり、存在が上書き・消滅していく。

---

## 2. 4つの公理（Axioms）

| 公理 | 内容 |
|------|------|
| **A** | 世界は整合性を優先する（矛盾が閾値を超えると収束が発生） |
| **B** | 整合性の獲得には歪みエネルギーが必要（無料ではない） |
| **C** | 最小修正で整合を取る（物理より意味・定義・関係の編集が安い） |
| **D** | PLは「人が世界を操作する力」ではなく「世界がその人をそう扱うルール」 |

## 5. 物語/音楽への使いどころ

### 物語
- **事件テンプレ**（v1.1 §8）：関係置換型、履歴再編集型、内面侵食型、完全消滅型
- **派閥抗争サイクル**（v1.3 §8）：AC接近→R派プロトコル→C派封鎖→A派救出→DI上昇→IO/IE進行→記録改稿→曖昧さ→さらなる実験

### 音楽
- **反復（ループ/リフ）**：自己アンカー、収束の反復
- **崩れ（ノイズ/転調/テンポ揺れ）**：歪債回収
- **置換（主旋律のすり替え/別テーマの侵入）**：IO-5
- **消滅（無音/フェードではなく"存在しなかった"終止）**：IE
