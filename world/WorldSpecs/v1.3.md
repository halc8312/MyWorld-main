# 世界仕様書 v1.3（綴合：派閥分裂／第二事故計画／A0・BLの介入）

## 1. 派閥分裂の前提（なぜ割れるのが“研究者崩れ”として自然か）
綴合は「現実が整合性に収束する」ことを受け入れた時点で、研究目的が必ず分裂する。

- **観測で満足する層**：モデル化・予測・論文化（内部報告）で到達したい  
- **制御に踏み込む層**：再現実験しない限り科学ではない、という強迫観念  
- **恐怖を抱く層**：事故の代償（IO/IE）を知ったことで、研究を止めたい／救いたい

加えてこの世界では、研究の失敗が「記録消失」ではなく**人間の上書き・消滅**なので、倫理ではなく“生存本能”で割れる。

---

## 2. 綴合の派閥：3極＋中間層（運用できるように固定）

### 2.1 派閥I：再現派（Repro / R派）
**目的**：第二事故を起こし、第二のA0（A0’）を作る。  
**信条**：「単発事故はデータではない。再現性こそ支配だ。」  
**リスク許容**：高（人員損耗やIEも“コスト”と扱う）

- 研究テーマ：アンカーの再設置、AC以外の周期の実現、分割付与の安定化
- 行動：DIを意図的に上げる、候補13歳を確保、観測と欠損を同時に設計
- 物語的役割：明確な敵にも、魅力的な狂気にもなる（冷たい合理）

### 2.2 派閥II：収容派（Contain / C派）
**目的**：現象を封じ、AC/IO/IEの被害を最小化しつつ、綴合を存続させる。  
**信条**：「A0は災害。観測し、隔離し、被害を抑える。」  
**リスク許容**：中（研究はするが、再現は避ける）

- 研究テーマ：歪度（DI）の低減、自己アンカー（SA）保全、観測固定（OL）の最適化
- 行動：保持者の隔離・保護（名目上）、情報統制、R派の暴走阻止
- 物語的役割：味方にも敵にもなる（“正しい”が冷酷）

### 2.3 派閥III：離反派（Abolish / A派）
**目的**：綴合の研究を止め、被験者・保持者を救出し、資料を破棄する。  
**信条**：「記述するほど世界を汚す。もうやめろ。」  
**リスク許容**：低〜中（ただし追い詰められると過激化）

- 研究テーマ：救出導線、SAの配布、上書き被害者の“核”の維持
- 行動：内部告発、データ破壊、被験者移送、R派施設破壊
- 物語的役割：主人公側に置きやすいが、やり方次第でDIを上げてしまう悲劇も作れる

### 2.4 中間層：便宜派（Prag / P層）
- 生き残りたい、金が欲しい、弱みを握られてる、など
- その場で派閥を変える（スパイ役に最適）
- 現場（Ops/Rec）の多くはここに落ちる

## 3. 派閥対立が“世界の収束”に与える影響（重要：設定が自己駆動する）
派閥抗争は単なる人間ドラマではなく、世界の整合性コストを増やす。

- R派：DIを上げる → 収束が頻発 → IO/IEが増える → 記録が壊れる → 研究がさらに過激化
- C派：OLを固める → 記録改稿は減るが、上書きが内面へ逃げる → “被害が見えない”
- A派：施設破壊・告発 → 観測が乱れる → DIが跳ねる → 最小修正で“関係”が破断しやすくなる

つまり、誰が正しくても地獄が深くなる。これがこの世界の強みです。

## 4. 「第一のA0（A0-1）」を派閥がどう扱うか（関与のさせ方）

### 4.1 A0-1の原則（世界観維持のための固定）
- A0-1は「会って話せば解決するボス」にはしない  
- A0-1は **所在不明／置換の可能性／認知されにくい**存在として扱う  
- ただし“痕跡”は残る：AC前後の異常、周辺でのDI急上昇、特定のテーマの反復

これで神秘性が保てます。

### 4.2 派閥ごとのA0-1観
- R派：A0-1は「初期条件」。再現すれば再生産可能  
- C派：A0-1は「災害源」。見つけたら隔離・封鎖  
- A派：A0-1は「被害者でもある」。だが接触は危険、という葛藤

### 4.3 A0-1が物語に関与する“現代的な出方”
- 直接登場ではなく、**A0-1の周囲一年の出来事がPL生成に滲む**ことで“存在感”が出る  
- あるいは、A0-1の痕跡（録音・メロディ・手書き）＝強烈なSAとして残り、保持者や研究者がそれに触れて収束が動く

> 音楽制作にも直結します：  
> 「A0-1の残したモチーフが世界の杭になってしまう」＝反復と変奏のホラー。

## 5. 保持者（BL）が綴合に関与してくるルート（仕様として成立させる）

保持者は基本「能力だと思っている」ので、関与の仕方を3本用意すると回しやすいです。

### 5.1 ルートA：回収（拉致ではなく“保護”名目）
- C派が「保護・治療・訓練」を提示し、保持者を囲う
- 保持者は能力を制御したいので乗りやすい
- だがOLが強まり、上書きが内面へ逃げていく（ホラー）

### 5.2 ルートB：協力（保持者側から接近）
- 生活が崩壊（IO-2〜3）し、本人が「説明できる相手」を求めて綴合へ辿り着く
- 綴合は“能力”という言葉を否定せず、少しずつPL概念へ誘導する
- 研究材料として扱われる危険と、唯一の理解者を得る救いが同居する

### 5.3 ルートC：対立（保持者が派閥抗争の鍵になる）
- R派は保持者を「第二事故の部品」にしたい  
- A派は保持者を「救う象徴」にしたい  
- C派は保持者を「封じるべき危険」にしたい  
→ どこに転んでも保持者は“能力バトル”ではなく“存在の取り合い”に巻き込まれる

## 6. 第二事故計画（A0’生成）の仕様：派閥で割れる“実験計画書”の骨子

### 6.1 第二事故の成功条件（綴合R派の仮説）
第二事故＝アンカー逆転（Anchor Inversion）の再現。必要条件は以下の同時満足：

1) **DIの局所過剰**（未整合の密度が高い）  
2) **観測の檻**（強い観測と弱い観測の混在で逃げ道を設計）  
3) **自己物語の臨界**（13歳、または同等の自己同一性臨界状態）  
4) **定義固定プロトコルの反復**（同一文言・同一儀式の反復で収束を誘導）  
5) **安全弁の破り方**（年次処理に落ちない“短周期編集”を引き起こす試み）

ここまでやると、R派はほぼ確実に人を壊します。

### 6.2 C派が恐れるポイント（仕様としての反論）
- A0’が生まれた瞬間、世界編集の中心が増える  
- するとAC/PLの体系自体が変わり、これまでのS1（年1神話）も崩れる可能性  
- 崩れた世界は「最小修正」で戻らない（不可逆の相転移）

### 6.3 A派の主張（仕様としての破壊動機）
- 第二事故の試行そのものがDIを上げ、無関係な市民にIO/IEが波及する
- 「研究を続ける限り、被害は統計的に増える」  
→ だからデータも施設も焼く、という過激化が起きる

## 7. “第一のA0”と“保持者”が将来的に交わるための仕掛け（面白さの核）
ここがあなたの「面白いかも」を、設定として確定できる部分です。

### 7.1 仕掛け1：A0-1の痕跡は“自己アンカー（SA）として最強”
- A0-1が残した音声・メロディ・言葉・署名は、世界の深層に刺さっている可能性がある  
- 保持者がそれに触れると、IOの進行が止まるのではなく「別様式に変わる」  
  - 例：消滅が置換に変わる／内面侵食が記録改稿に逃げる  
→ “救いにも呪いにもなる杭”

### 7.2 仕掛け2：保持者のPLが、A0-1の一年の出来事テーマと共鳴する年がある
- ある年のPL群が同じ“テーマ”を帯びる（例：名前、呼称、関係、真実、観測）  
- 綴合はそれを「A0-1が今年それに晒された」兆候として追跡する  
- 結果、保持者が捜索装置になる（本人は知らない）

### 7.3 仕掛け3：複数付与年は、A0-1の近傍で起きやすい
- A0-1周辺のDIが高いと、ACの“分割付与”条件が成立しやすい  
- だから「複数付与が疑われる年」を追うと、A0-1に近づく  
→ 研究者崩れが大好物な、統計追跡型ホラーが可能

## 8. 作品運用のための“固定テンプレ”：綴合内部抗争の基本サイクル
物語を量産できるよう、抗争が自走するテンプレを置きます。

1) ACが近づく（DI上昇）  
2) R派が候補を確保し、プロトコルを仕掛ける  
3) C派が封じに動き、OLを設計して“事故を回避”しようとする  
4) A派が候補救出・施設破壊を計画  
5) その全てがDIを上げ、世界が収束し、誰かのIO/IEが進む  
6) 結果が「最小修正」で記録改稿され、責任の所在が曖昧になる  
7) 研究者崩れは曖昧さに耐えられず、さらに強い観測と実験に走る
