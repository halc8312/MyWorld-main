# 綴合（てつごう）/ Tetsugo

## 概要

**綴合**は、「整合性の挙動を記述し、再現可能な操作へ落とすための集団」を自称する地下研究組織。
研究者崩れ（元学者、元医療従事者、元データ工学者など）を中心に構成される。

- **成立**：AC-0021 年
- **分裂**：AC-0031 年（複数付与年の隠蔽失敗による派閥分裂）
- **現在**：3派閥（R派/C派/A派）＋便宜派（P層）に分裂した状態で継続

---

## 組織理念（内部で共有される短文）

壁に貼ってあるレベルの原理信条：

1. **現実は合意と記録に収束する**
2. **改変は最小修正で行われる**
3. **観測は武器であり麻薬である**（強めると別層に逃げる）
4. **倫理はモデルを歪ませる**（＝"ノイズ"扱い）

> この冷たさが、ホラーのベースになる。

---

## 表の顔（合法の顔）

綴合は複数の表看板を持ち、合法的な活動を装う：

| 種類 | 例 | 役割 |
|------|----|------|
| 心理支援NPO | 青少年サポートセンター | 候補13歳への接触導線 |
| 教育支援財団 | ○○教育振興財団 | 塾、合宿、奨学金を通じた囲い込み |
| セキュリティ研究法人 | ○○情報セキュリティ研究所 | 観測技術、データ統合の研究名目 |
| 治験委託会社 | ○○メディカルリサーチ | 心理プロトコルの実施名目 |
| 探偵業 | ○○総合調査 | 保持者追跡、工作活動 |

---

## 班編成（現在の構造）

綴合は企業的ピラミッドではなく、**研究室の集合体**に近い構造を持つ。

| 班 | 略称 | 役割 | 典型的な人材 |
|----|------|------|--------------|
| **評議** | Council | 方針と資金配分。現場に降りない | 元教授、元官僚、資産家 |
| **主任研究** | PI | 各プロジェクト責任者。実験設計と成果物 | 元研究者、元医師 |
| **観測班** | Obs | 記録収集、同期、改ざん耐性設計、死角作り | データ工学、セキュリティ |
| **プロトコル班** | Proto | 心理手順書、質問票、台本、同意文書（偽） | 心理士、脚本家 |
| **工作班** | Ops | 外部対応、攪乱、連絡、搬送、証人処理 | 元警備、元探偵、フリー |
| **保全班** | Sec | 内部リーク対策、裏切り処理、端末管理 | 元情報機関、元IT |
| **回収班** | Rec | 13歳候補の導入導線（塾、相談窓口、里親） | 教師、福祉、スカウト |

## 施設構成（標準モデル）

都市近郊の雑居ビル、または郊外の研修施設を使用。

| フロア | 名称 | 機能 |
|--------|------|------|
| **F1** | 受付・カウンセリング（表） | 面談室、待合、簡易検査、同意書取得、スマホ預かり |
| **B1** | 観測制御フロア（裏） | カメラ統合、音声、時刻同期、ログ収集、死角作成装置 |
| **B2** | プロトコル実施フロア（裏） | 被験者室（生活可能）、面接室（同一質問反復）、記録室 |
| **B3** | 保全・焼却・物品庫（裏） | 物理媒体保管庫、焼却炉、非合法物隔離棚 |
| **オフサイト** | 分散倉庫 | 物理媒体の分散保管（単一障害点回避） |

## 派閥（AC-0031 分裂後）

### R派（再現派 / Repro）

| 項目 | 内容 |
|------|------|
| **目的** | 第二事故を起こし、第二の A0（A0'）を作る |
| **信条** | 「単発事故はデータではない。再現性こそ支配だ」 |
| **リスク許容** | 高（人員損耗やIEも"コスト"と扱う） |
| **研究テーマ** | アンカーの再設置、AC以外の周期の実現、分割付与の安定化 |
| **行動** | DI を意図的に上げる、候補13歳を確保、観測と欠損を同時に設計 |
| **物語的役割** | 明確な敵にも、魅力的な狂気にもなる（冷たい合理） |

**典型的な R派メンバー**：
- 元物理学者。「世界は方程式で記述できる」と信じている
- AC-0031 で「データが取れた」と興奮した側
- 目的のためなら自分自身も実験台にできる

---

### C派（収容派 / Contain）

| 項目 | 内容 |
|------|------|
| **目的** | 現象を封じ、AC/IO/IE の被害を最小化しつつ、綴合を存続させる |
| **信条** | 「A0は災害。観測し、隔離し、被害を抑える」 |
| **リスク許容** | 中（研究はするが、再現は避ける） |
| **研究テーマ** | 歪度（DI）の低減、自己アンカー（SA）保全、観測固定（OL）の最適化 |
| **行動** | 保持者の隔離・保護（名目上）、情報統制、R派の暴走阻止 |
| **物語的役割** | 味方にも敵にもなる（"正しい"が冷酷） |

**典型的な C派メンバー**：
- 元精神科医。保持者を「患者」として見ている
- AC-0031 で「これ以上 IE を出してはならない」と主張した側
- 保護を名目に、実際は監視と管理を行う

---

### A派（離反派 / Abolish）

| 項目 | 内容 |
|------|------|
| **目的** | 綴合の研究を止め、被験者・保持者を救出し、資料を破棄する |
| **信条** | 「記述するほど世界を汚す。もうやめろ」 |
| **リスク許容** | 低〜中（ただし追い詰められると過激化） |
| **研究テーマ** | 救出導線、SA の配布、上書き被害者の"核"の維持 |
| **行動** | 内部告発、データ破壊、被験者移送、R派施設破壊 |
| **物語的役割** | 主人公側に置きやすいが、やり方次第で DI を上げてしまう悲劇も作れる |

**典型的な A派メンバー**：
- 元回収班。囲い込んだ13歳の一人と心を通わせてしまった
- AC-0031 で「BL-0031B は存在していた。覚えている」と主張した側
- 「BL-0031B 文書」を密かに保管している

---

### P層（便宜派 / Prag）

| 項目 | 内容 |
|------|------|
| **目的** | 生き残りたい、金が欲しい、弱みを握られている |
| **信条** | なし（状況による） |
| **リスク許容** | 様々 |
| **行動** | その場で派閥を変える |
| **物語的役割** | スパイ役に最適、現場（Ops/Rec）の多くがここに落ちる |
