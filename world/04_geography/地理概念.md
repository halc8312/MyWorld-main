# 地理概念（Geography Concepts）

## 概要

本世界の地理は、物理的な地形よりも**歪度（DI）の分布**によって特徴づけられる。
DI が高い場所では現象が起きやすく、低い場所では安定している。

> [!NOTE]
> 具体的な国名、都市名は物語ごとに調整可能。
> 本ドキュメントでは概念的な地理を定義する。

---

## 1. 歪度（DI）の地域差

### 1.1 DI に影響する要因

| 要因 | DI への影響 | 例 |
|------|------------|-----|
| **未解決事件** | 上昇 | 真相未確定の事件現場 |
| **集団の分断** | 上昇 | 同じ事実への解釈が割れる地域 |
| **偽情報の流通** | 上昇 | フェイクニュースが多い地域 |
| **記録の消失・改ざん** | 上昇 | 戦災、災害、行政の怠慢 |
| **観測の分散** | 上昇 | 監視は多いが統合されていない都市 |
| **観測の固定** | 低下 | 強い合意形成がある地域 |
| **孤立・過疎** | 変動 | 観測が少ないため安定しやすいが、一度崩れると一気に変動 |

### 1.2 典型的な DI 分布

| 地域タイプ | DI | 特徴 |
|------------|-----|------|
| **大都市中心部** | 高 | 監視多いが分散、情報過多、匿名性高い |
| **郊外住宅地** | 中 | 観測は中程度、コミュニティが存在 |
| **地方都市** | 中〜低 | 観測少ないが固定されやすい |
| **過疎地域** | 低（変動リスクあり） | 観測非常に少ない、一度崩れると急変 |
| **ネット空間** | 高 | 偽情報と真実が混在、合意形成が不安定 |
| **未解決事件現場** | 極高 | 真相未確定により DI が局所的に上昇 |
| **A0-1 の近傍** | 極高 | 複数付与が起きやすい（v1.3 §7.3） |

## 2. 綴合の拠点

### 2.1 拠点の選定基準

綴合は以下の基準で拠点を選ぶ：

1. **DI が中程度**：高すぎると収束が予測不能、低すぎると現象が観測しにくい
2. **表の顔が成立する**：NPO、研修施設、クリニックとして不自然でない
3. **交通の便**：回収班の活動に必要
4. **記録の管理が容易**：観測固定（OL）を設計しやすい
5. **分散できる**：単一障害点を避けるため、複数拠点を持つ

### 2.2 典型的な拠点タイプ

| タイプ | 立地 | 表の顔 | 特徴 |
|--------|------|--------|------|
| **都市近郊雑居ビル** | 大都市の郊外 | 心理支援NPO、セキュリティ研究 | アクセス良好、匿名性高い |
| **郊外研修施設** | 住宅地と田舎の境界 | 教育支援財団、合宿所 | 隔離しやすい、観測制御しやすい |
| **メンタルクリニック併設** | 都市部 | クリニック、治験委託 | 医療名目で長期滞在可能 |
| **分散倉庫** | 各地 | レンタル倉庫、貸金庫 | 物理媒体の分散保管 |

### 2.3 施設の内部構成（再掲）

| フロア | 名称 | 機能 |
|--------|------|------|
| F1 | 受付・カウンセリング（表） | 面談室、待合、同意書取得 |
| B1 | 観測制御フロア（裏） | カメラ統合、ログ収集、死角作成 |
| B2 | プロトコル実施フロア（裏） | 被験者室、面接室、記録室 |
| B3 | 保全・焼却・物品庫（裏） | 物理媒体保管、焼却炉 |

## 3. 特殊な地理的概念

### 3.1 「A0-1 の近傍」

v1.3 §7.3 に基づき、A0-1 の近傍は特殊な地理的性質を持つ：

- **複数付与が起きやすい**：A0-1 周辺の DI が高いと、AC の「分割付与」条件が成立しやすい
- **追跡の手がかり**：複数付与が疑われる年を追うと、A0-1 に近づく
- **不安定**：A0-1 の近傍では収束が予測不能になりやすい

> ただし、A0-1 の所在は不明であり、「近傍」の特定は困難。
> 複数付与の発生地点を逆算することで推定する（綴合 R派の手法）。

### 3.2 「歪む実験」の跡地

最初の事故が起きた場所：

- **現在の状態**：火災で焼失、または売却・再開発された可能性
- **DI**：極高（ただし世界の収束により「何もなかった場所」として整合されている可能性）
- **物理的痕跡**：残っているかは未確定（物語のフック）
- **綴合の関心**：R派は跡地に何か残っていると信じている、A派は封印すべきと考えている

### 3.3 「BL-0031B の痕跡」がある場所

派閥分裂事件で IE となった BL-0031B に関連する場所：

- **BL-0031B が住んでいた地域**：現在は「その人がいなかったこと」に整合されている
- **A派が保管する資料の保管場所**：物理的に分散されている
- **触れると収束が不安定になる**：強い SA 残滓として機能する可能性

## 4. 地理と物語

### 4.1 DI による物語の舞台設定

| DI | 舞台として適した物語 |
|-----|----------------------|
| 高 | 派手な現象、複数の保持者、カオス、都市伝説 |
| 中 | バランスの取れた展開、探偵もの、学園もの |
| 低 | じわじわ進行するホラー、孤立した舞台、クローズドサークル |

### 4.2 綴合拠点を舞台にする場合

- **F1（表）**：一般人が来る場所、導入に適している
- **B1-B3（裏）**：クライマックス、真相の暴露に適している
- **分散倉庫**：追跡パート、資料発見に適している

### 4.3 移動と距離

- 現代日本を想定した場合、公共交通機関と自動車が主な移動手段
- 綴合の回収班は偽装車両（福祉車両、医療車両）を使用
- 保持者の逃走距離は、IO の進行により「居場所が曖昧になる」ことで変動
